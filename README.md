
# Project Description

This project involves developing a **fully-functional frontend website** that comprehensively details the **Pharmacy Department standards** in a hospital setting.

The website serves as a complete digital resource explaining every aspect of pharmacy operations, systems, and standards used in hospitals. 

Key topics covered:
- Pharmacy Information Systems (PIS)
- Medication management workflows
- Drug dispensing, inventory, and administration processes
- Integration with hospital-wide systems
- Compliance with national and international healthcare standards

---

# Key Features of the Website

- **Modern, Responsive UI/UX** designed for clarity and professionalism
- **Interactive sections** for exploring pharmacy workflows
- **Detailed software architecture diagrams**
- **Process flowcharts** showing medication ordering, dispensing, and administration
- **Interoperability section** explaining connections with standards such as:
  - HL7
  - FHIR
  - DICOM
  - ICD-10
  - HIPAA / other regulatory standards
- **Comprehensive documentation** of the Pharmacy Department standard

---

# Technologies Used

- **Frontend**: HTML, CSS, JavaScript

---

# How to Run / View the Website

1. Clone the repository
2. Open `index.html` in your browser (static frontend)
